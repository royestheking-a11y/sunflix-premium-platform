# SUNFLIX Color Update - Complete Replacement Guide

## OLD COLORS (Masalafun) → NEW COLORS (SUNFLIX)

### Primary Gradients
- `from-[#6C00FF] to-[#00FFD1]` → `from-[#FF7E00] to-[#6C00FF]`
- `from-[#6C00FF] to-[#8B3FFF]` → `from-[#FF7E00] to-[#FFB800]`

### Accent Colors
- `text-[#00FFD1]` → `text-[#FFB800]`
- `text-[#6C00FF]` → `text-[#FF7E00]` or `text-[#FFB800]`
- `border-[#6C00FF]` → `border-[#FF7E00]`
- `border-[#00FFD1]` → `border-[#FFB800]`

### Background Glows
- `from-[#00FFD1]/20` → `from-[#FFB800]/20`
- `to-[#00FFD1]/20` → `to-[#6C00FF]/20`

### Shadow Effects  
- `shadow-[#6C00FF]/50` → `shadow-[#FF7E00]/50`
